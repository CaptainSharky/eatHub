## Getting started

### Prerequisites
* Xcode 15+
* Swift 6+
* XcodeGen or Homebrew

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/CaptainSharky/eatHub.git
   ```
2. Find the project folder
   ```sh
   cd eatHub
   ```
3. Generate .xcodeproj file with Homebrew
   ```sh
   sh start.sh --full
   ```
